# Recicla - Belford Roxo

O **Reciclar** é uma aplicação mobile desenvolvida como projeto de extensão para o curso de Análise e Desenvolvimento de Sistemas. O objetivo é facilitar a logística reversa de lixo eletrônico, conectando os cidadãos de Belford Roxo a pontos de coleta autorizados.

## Funcionalidades
- **Busca por Material:** Identifica o ponto de coleta ideal conforme o resíduo (ex: eletrônicos, baterias).
- **Navegação Inteligente:** Integração direta com Google Maps para traçar rotas.
- **Educação Ambiental:** Guia prático sobre o impacto de metais pesados (Chumbo, Mercúrio, Cádmio) conforme normas do INEA.

## Download do Aplicativo
Para instalar o aplicativo no seu dispositivo Android:
1. Vá até a seção [Releases](https://github.com/PedroCassiano1996/Projeto-de-extensao-recicla/releases/tag/v1.0.0) deste repositório.
2. Baixe a versão mais recente do arquivo `Reciclar.apk`.
3. Instale no seu Android (autorize "fontes desconhecidas" se necessário).

## Tecnologias
- Linguagem: Python 3
- Interface: Kivy Framework
- Ferramenta de Build: Buildozer (Android)
